Solstice Artella Uploader

This is the main tool used to upload, lock and unlock files of multiple Artella files at once.

Solstice Artella Uploader

How it works?

  1. You select a path inside your local Solstice Project in your local drive. The tool will show you all the files located inside that folder.
  2. In the files list you need to check/uncheck the files you want to work with.

Now you have 3 different options:

Name Description
Lock File Lock all the checked files in Artella server.
Unlock File Unlock all the checked files in Artella server.
Make New Version Uploads a new version into Artella server of all the checked files.

Important

When uploading new version of files that are already in Artella server, you need to make sure that all the files are locked.
